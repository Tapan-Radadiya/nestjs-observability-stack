# 🚀 NestJS Observability with Prometheus, Grafana & Loki

This project demonstrates how to implement **end-to-end observability** in a NestJS application using Prometheus, Grafana, and Loki, along with custom metrics.

---

## ✨ Features

- 📊 Prometheus metrics integration
- 📈 Grafana dashboards for visualization
- 📜 Loki for centralized logging
- ⚙️ Custom metrics (request count & latency)
- ⏱️ P95 latency tracking
- 🚨 Error rate monitoring (including 5XX responses)
- 🔄 Interceptor-based request tracking
- 📦 Ready-to-use Grafana dashboard JSON
- 🧪 Postman collection included for API testing

---

## 🛠️ Tech Stack

- NestJS  
- Prometheus  
- Grafana  
- Loki  
- prom-client
